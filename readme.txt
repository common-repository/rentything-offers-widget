Offers from Rentything.com

== Description == 

This is a widget that displays a user's offers from Rentything.com to help promote their offers.

== Installation == 

1. Download the plugin
2. Upload the rentything-widget folder to your site's  wp-content/plugins directory
3. Activate the plugin in the WordPress Admin Panel 
4. Add the widget! 
5. Fill out the username and the number of offers you want displayed on the widget form.
